# Privacy Policy

This is a single-user personal AI assistant deployment.

## Data collection
This application accesses Gmail and Google Calendar data only on behalf of the
authorized user. No data is collected, transmitted, or shared with third parties
beyond the AI inference provider used to generate responses.

## Data storage
All data remains on the user's personal hardware. No cloud database, analytics,
or telemetry is used.
